## Install

Install the latest [go-libp2p](https://github.com/libp2p/go-libp2p/) binary:

```sh
# Install globally
> npm install -g go-libp2p
> libp2p version
libp2p version v0.7.0

# Install locally
> npm install go-libp2p
> ./node_modules/.bin/libp2p
libp2p version v0.7.0
```
